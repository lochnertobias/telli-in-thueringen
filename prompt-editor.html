So funktioniert der Editor

Mit diesem Editor können Sie ganz einfach Sammlungen von Arbeitsanweisungen (sogenannte "Prompts") für KI-Modelle erstellen, anpassen und verwalten. Die Bedienung ist unkompliziert und erfordert keine Programmierkenntnisse.

1. Listen laden oder neu erstellen

Über die Symbole in der Werkzeugleiste oben links haben Sie folgende Möglichkeiten:

  • Laden (Ordner-Symbol): Wählen Sie eine bestehende Promptliste aus der Server-Liste aus oder laden Sie eine Datei von Ihrem PC hoch.
  • Neu (Plus-Symbol): Beginnen Sie mit einem komplett leeren Dokument, für das Sie selbst ein Fach festlegen können.

2. Prompts bearbeiten

Sobald eine Liste geöffnet ist, können Sie die einzelnen "Kärtchen" bearbeiten. Sie können Titel, Zielgruppe (Stufe) und Tags anpassen oder den eigentlichen Text des Prompts verändern. Über die Buttons am unteren Bildschirmrand lassen sich jederzeit neue, leere Kärtchen oder vorgefertigte Mustereinträge hinzufügen.

WICHTIGER HINWEIS zum Speichern

Dieser Editor arbeitet zu Ihrem Schutz und für höchste Datensicherheit ausschließlich lokal in Ihrem Internetbrowser. Das bedeutet:

  • Wenn Sie auf "Speichern" (Disketten-Symbol) klicken, wird Ihre Arbeit als Datei direkt auf Ihren Computer heruntergeladen (in der Regel in Ihren "Downloads"-Ordner).
  • Es findet keine automatische Speicherung im Internet oder auf dieser Webseite statt.
  • Ihre Änderungen sind für andere Nutzer erst sichtbar, wenn Sie Ihre gespeicherte Datei an den Webseiten-Administrator weiterleiten, damit dieser die neuen Daten in das System einpflegt.

Bitte denken Sie also immer daran, Ihre fertige Arbeit über den Speichern-Button auf Ihrem eigenen PC zu sichern, bevor Sie das Browserfenster schließen!
